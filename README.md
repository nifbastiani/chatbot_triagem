# chatbot_triagem
## Requisitos do Projeto

O chatbot de triagem foi desenvolvido com o objetivo de ser um **programa acessível e funcional sem necessidade de internet**, permitindo que **pessoas em situação de vulnerabilidade** ou que vivem em **áreas rurais** possam receber um **diagnóstico preliminar** de forma simples e rápida.

### Requisitos principais:
- Deve funcionar **offline**, sem depender de conexão com a internet.  
- Deve possuir **interface simples**, permitindo o uso por qualquer pessoa, mesmo com pouca familiaridade com tecnologia.  
- Deve realizar **perguntas sobre sintomas** e, com base nas respostas, **identificar a possível doença** (como dengue, gripe ou covid-19).  
- Deve **exibir orientações** e **sugestões de tratamento inicial** conforme o diagnóstico estimado.  
- Deve **alertar o usuário** quando houver necessidade de procurar um hospital ou atendimento médico presencial.  
- Deve contribuir para a **inclusão e acesso à informação em saúde**, reduzindo os riscos causados pela automedicação e pela falta de diagnóstico adequado.
